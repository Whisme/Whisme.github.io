1.安装模块

  • 书本上说使用pip install urllib3的指令在CMD命令中安装该库
  • 其实可以直接在pycharm中安装urllib3库

  • 输入import urllib3 下方会有红色波浪线
  • 点击install package urllib3
  • 也可以鼠标放到 urllib3旁边——使用快捷键Alt+Shift+Enter

2.没有红色波浪线的情况下安装模块

  • 由于安装pycharm版本问题，可能出现不提示红色波浪线等情况
  • 打开pycharm工具栏——file——settings
  • 搜索Python Interpreter
  • 双击pip——然后在搜索框中搜索想要安装的urllib3——点击左下角Install package安装
  • 其他任意模块（也可以叫包）都可以使用该方法安装

计算机网络原理（笔记）

计算机网络功能

  • 基本功能：数据通信
  • 主要功能：资源共享

按地理范围划分

  • 局域网（Local area networks）范围：<10KM
  • 广域网（Wide area networks）范围：>100KM
  • 城域网（Metropolitan area networks）范围：<100KM
  • 个人区域网（PAN）范围：10米内

计算机网络按通讯所使用介质分类

  • 有线网
  • 无线网

计算机网络按使用网络对象

  • 专用网络
  • 公共网络

拓扑结构

  • 最常用的结构：星型
  • 很少用的：环形
  • 淘汰的：总线
  • 广域网最常用：网状
  • 局域网最常用：星型

计算机网络性能指标

  • 速率
  • 带宽
  • 吞吐量
  • 时延
